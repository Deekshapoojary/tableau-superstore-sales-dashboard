# 📊 Tableau Superstore Sales Dashboard

## Overview

This interactive Tableau dashboard analyzes Superstore sales performance across categories, products, regions, and customer segments. It provides business insights through interactive filters, KPIs, maps, and trend analysis.

---

## Dashboard Preview

![Dashboard](Images/Dashboard%20Tableau.png)

---

## Business Objectives

- Monitor Total Sales
- Analyze Profit Margin
- Track Return Rate
- Compare Product Categories
- Analyze Customer Segments
- Monitor Regional Performance
- Identify Top Products
- Analyze Monthly Sales Trends

---

## Tools Used

- Tableau Desktop
- Tableau Public
- Interactive Dashboards
- Calculated Fields
- Parameters
- Filters
- Maps
- KPI Cards

---

## Dashboard Features

- Interactive Filters
- KPI Summary Cards
- Geographic Sales Map
- Product Category Analysis
- Customer Segment Analysis
- Sales Trend Analysis
- Top Products Ranking
- Dynamic Dashboard Interactions

---

## Key Insights

- Consumer segment contributes over 50% of total sales.
- Technology is the highest-performing product category.
- California generates the highest sales among all states.
- Sales show an upward trend over multiple years.
